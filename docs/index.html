About This Project

This dashboard tracks the reproducibility of official Debian Docker images. We rebuild images using Debuerreotype and verify bit-for-bit reproducibility by comparing SHA256 checksums.

Why reproducibility matters: Reproducible builds prove supply chain integrity and allow independent verification that published binaries match their source code.
